# NYPD Year-to-Date Arrests Investigatory Analysis  
**COOP Group Project | September 2025**  
**Source: NYPD Open Data**

## üìå Project Overview
This project investigates arrest trends in New York City using publicly available NYPD Open Data.  
We analyzed **155,000+ arrest records** to uncover neighborhood-level crime concentrations and year-over-year shifts in arrest categories.  

The analysis was part of a **COOP Group Project (September 2025)** aimed at building data cleaning, analysis, and visualization skills for real-world datasets.
